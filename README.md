### Bug Reports

1. Make sure that this problem happens when using GridHax **on its own**, if so, then you should open an Issue for a Bug Report.
   - If the problem only happens when another mod is installed, ask in the Discord server and I will inform you if you should open an Issue.
2. Make sure you are on the latest version of GridHax (and the other mod, if applicable).
3. Collect detailed steps to reproduce the bug.
4. [Report the bug][new bug] with an accurate and helpful title and description, make report here https://github.com/Grid-Storm-INCORPERATED/GridHax-git-issues/issues
5. Wait patiently for a response. If it has been over 48 hours with no response to your issue, please contact a Support member in the GridHax Discord.

### Suggestions & Enhancements

1. Look around the client and be **absolutely sure** it does not already exist and that its functionality cannot be achieved within GridHax.
2. Collect detailed information on what exactly you want to be added. Provide examples and use cases, if possible.
3. [Submit the suggestion][new suggestion] with aforementioned information.
4. Wait patiently for a response. If it has been over 96 hours (4 days) with no response to your suggestion, please contact a support member in the GridHax Discord.
